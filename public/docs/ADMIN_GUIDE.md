# Admin Panel Guide

## Access & Roles

You need **Moderator** or higher to access the admin panel. Role hierarchy:

- **User** — Basic site access
- **Member** — Member privileges  
- **Moderator** — Dashboard access, basic settings
- **Admin** — Full control (this is you)

---

## Deck Management

### Import New Decks

**Single deck:** Click Import Deck → paste Moxfield URL → Import

**Bulk import from bookmarks:** Click Fetch All → wait a few minutes → page auto-refreshes

### Update Existing Decks

Click Update All Decks to refresh view counts and metadata from Moxfield.

### Edit a Deck

Find deck → Edit → modify name, description, video links, etc. → Save Changes

---

## User Management

Search users by email. For each user you can see their Patreon tier, credits balance, and submission history.

**Change role:** Find user → click role dropdown → select new role (applies immediately)

**Add user:** Click Add User → enter email → set role → Add User

---

## Submissions Workflow

Submissions page shows pending requests, oldest first.

1. **Start** — Click when you begin working on a deck (moves to in-progress)
2. **Finish** — Click when complete (links deck to user's profile)
3. **Reject** — Use if submission can't be fulfilled

---

## Products

Add/edit store products linked to Fourthwall.

**Add:** Click Add Product → fill in name, link, image, category, sort order → Add Product

**Edit:** Modify fields inline → Save

**Sort order:** Lower numbers display first

---

## Credits & Patreon Tiers

Found in Site Settings → Credits Tab.

### Benefits Matrix

Controls how many credits each Patreon tier gets monthly:

| Tier | Deck Credits | Roast Credits |
|------|--------------|---------------|
| Emissary | 0 | 1 |
| Duke | 1 | 1 |
| ArchMage | 2 | 1 |
| Wizard | 3 | 1 |

Edit the matrix → Save Benefits Matrix. Credits distribute automatically each month.

---

## Site Settings

Site Settings tab lets you manage videos, social links, and other site-wide config. Add/edit values as needed, then Save Changes.

---

## Quick Reference

| Task | Where |
|------|-------|
| Import decks from Moxfield | Deck Management → Fetch All |
| Process a submission | Submissions → Start → (build deck) → Finish |
| Change user role | User Management → search → role dropdown |
| Add store product | Products → Add Product |
| Adjust tier credits | Site Settings → Credits → Benefits Matrix |
