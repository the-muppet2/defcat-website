# Data Reference

Overview of data stored in DefCat DeckVault.

## User Data

### Profiles
User accounts with Patreon integration:
- Email address
- Patreon ID and tier
- Moxfield username
- Site role (User, Moderator, Admin)
- Account creation date

### Credits
Monthly credit allocation per user:
- Deck credits balance
- Roast credits balance
- Last distribution date

---

## Deck Data

### Imported Decks
Deck metadata imported from Moxfield:
- Deck name and author
- Commander(s) and color identity
- Format and power bracket
- Card counts (mainboard, sideboard)
- View/like/comment counts
- Import and last update timestamps

### Card Cache
MTG card data cached from Scryfall:
- Card name and type
- Mana cost and colors
- Card text and rarity
- Set information
- Image URLs
- Price data

---

## Submission Data

### Deck Submissions
User deck submission requests:
- Submitter info (email, Discord, Moxfield username)
- Patreon tier at time of submission
- Submission type (Mystery Deck or Custom Build)
- Preferences (commander, colors, bracket, budget, theme)
- Status (pending, in_progress, completed, rejected)
- Timestamps

---

## Site Configuration

### Products
Store product listings:
- Product name and description
- Link and image URL
- Category and sort order
- Active/inactive status

### Site Config
Key-value configuration settings:
- Video links
- Social media URLs
- Feature flags
- General settings
